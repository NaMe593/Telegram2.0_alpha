# 🚀 НАЧАЛО РАБОТЫ - За 5 минут

## Что было создано?

Теперь ваше приложение умеет:
- 📡 Получать чаты и сообщения с сервера (REST API)
- 💾 Работать offline если сервер недоступен
- 🔄 Отправлять/получать данные асинхронно
- 🎯 Автоматически fallback на локальные данные

---

## ⚡ Быстрый старт (5 минут)

### 1️⃣ Откройте терминал в папке проекта
```bash
cd burgerGram
```

### 2️⃣ Установите серверные зависимости (если ещё нет)
```bash
npm install express cors
```

### 3️⃣ Запустите сервер в отдельном терминале
```bash
node server.js
```
Результат:
```
Сервер запущен на http://localhost:3000
```

### 4️⃣ В другом терминале запустите React
```bash
npm run dev
```
Результат:
```
http://localhost:5173
```

### 5️⃣ Откройте приложение в браузере
```
http://localhost:5173
```

✅ **Готово!** Приложение работает с сервером!

## 🔌 Как это работает?

```
┌─────────────────────────────────────────────────┐
│ React App (http://localhost:5173)               │
│                                                  │
│  App.jsx                                         │
│  ├─ useEffect: loadInitialData()                │
│  ├─ fetchChats() ──┐                            │
│  └─ fetchMessages() │                           │
└────────────────────┼──────────────────────────┘
                    │ HTTP requests
                    ▼
        ┌──────────────────────────────┐
        │ Node.js Server (localhost:3000)
        │ server.js (Express API)
        │
        │ GET    /api/chats
        │ GET    /api/chats/:id/messages
        │ POST   /api/chats/:id/messages
        │ POST   /api/chats
        │ DELETE /api/chats/:id
        └──────────────────────────────┘

⚠️ Если сервер падает → используются локальные данные
```

## 💡 Основные методы

### Загрузка (асинхронно)
```javascript
const chats = await chatService.fetchChats();
const messages = await chatService.fetchMessages(1);
```

### Отправка
```javascript
await chatService.sendMessage(1, 'me', 'Hello!', '14:00');
```

### Создание
```javascript
await chatService.createChat('John', 'Hey', 5, 'img.png');
```

### Получение локально
```javascript
const chats = chatService.getChats(); // Без асинхроуса
```

## 🛠️ Что делать, если...

### ❌ "Cannot GET /api/chats"
✅ **Решение:** Сервер не запущен  
```bash
node server.js
```

### ❌ "CORS error"
✅ **Решение:** Проверьте что сервер запущен на порту 3000

### ❌ Данные не загружаются
✅ **Решение:** Откройте консоль (F12) и посмотрите ошибки

### ❌ Хочу использовать свой API
✅ **Решение:** Измените URL в App.jsx:
```javascript
const chatService = new ChatService('http://my-api.com/api');
```

## 📊 Структура запроса

### Сохранение сообщения:

**React:**
```javascript
await chatService.sendMessage(1, 'me', 'Hello!', '14:00', null)
```

**Network запрос:**
```
POST http://localhost:3000/api/chats/1/messages
Content-Type: application/json

{
  "from": "me",
  "text": "Hello!",
  "time": "14:00",
  "avatar_id": null
}
```

**Сервер обработает и вернёт:**
```json
{
  "id": 3,
  "from": "me",
  "name": "You",
  "text": "Hello!",
  "time": "14:00",
  "avatar_id": null
}
```
